# Escudo Naval | EulerGames

Juego web educativo de intercepción de misiles en un plano cartesiano.

El jugador protege un barco fijo en el mar. Un misil enemigo entra desde el borde derecho con trayectoria, punto de salida, punto de impacto y velocidad conocidos. El jugador debe ajustar el ángulo del misil defensor y disparar en el momento correcto para interceptarlo.

## Cómo jugar

- Ajusta el ángulo con los botones o con el teclado.
- Presiona **Espacio** o el botón **Disparar** cuando el misil enemigo ya esté en escena.
- Presiona **Enter** para pausar o reanudar.
- Activa el sonido manualmente con el botón **Activar sonido**.

## Controles

- Flecha izquierda: disminuir 1°
- Flecha derecha: aumentar 1°
- Flecha abajo: disminuir 0.1°
- Flecha arriba: aumentar 0.1°
- Espacio: disparar
- Enter: pausar/reanudar

## Conceptos físicos y matemáticos

- Movimiento rectilíneo uniforme
- Posición, velocidad y tiempo
- Vectores en el plano
- Ángulo de disparo
- Intercepción de trayectorias

## Publicación en GitHub Pages

Este proyecto no requiere instalación ni dependencias. Basta con subir el archivo `index.html` al repositorio y activar GitHub Pages desde la rama principal.
